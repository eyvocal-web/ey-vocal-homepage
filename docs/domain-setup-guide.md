# EY보컬스튜디오 커스텀 도메인 설정 가이드

대상 프로젝트: `ey-vocal-homepage`

현재 기본 주소:
- `https://ey-vocal-homepage.vercel.app/`

## 1. 준비물

- Vercel 프로젝트 접근 권한
- 도메인 구매처(가비아, 후이즈, 카페24, Cloudflare 등) 관리자 권한
- 최종 사용할 대표 도메인 결정
  - 예시: `eyvocalstudio.com`
  - 예시: `www.eyvocalstudio.com`

## 2. 권장 도메인 구조

- 대표 도메인 1개를 정합니다.
- 나머지 도메인은 대표 도메인으로 리다이렉트합니다.

권장 예시:
- 대표: `www.도메인`
- 보조: 루트 도메인(`도메인`) → 대표 도메인으로 리다이렉트

또는
- 대표: 루트 도메인(`도메인`)
- 보조: `www.도메인` → 대표 도메인으로 리다이렉트

중요:
- 대표 도메인은 1개만 유지해야 검색엔진과 공유 링크가 안정적입니다.

## 3. Vercel에서 도메인 추가

1. Vercel 대시보드 접속
2. `ey-vocal-homepage` 프로젝트 선택
3. `Settings` → `Domains`
4. 사용할 도메인 입력 후 추가
5. Vercel이 보여주는 DNS 레코드를 그대로 복사

주의:
- DNS 값은 시점과 도메인 구조에 따라 달라질 수 있으므로, 문서에 적힌 예시보다 Vercel 대시보드에 표시되는 값을 우선합니다.

## 4. DNS 설정

도메인 구매처 관리자 화면에서 Vercel이 안내한 값을 등록합니다.

일반 흐름:
- 루트 도메인: A 레코드 또는 ALIAS/ANAME 방식 사용
- `www` 서브도메인: CNAME 방식 사용

체크 포인트:
- 기존에 연결된 오래된 A/CNAME 레코드가 있으면 정리
- 프록시 기능(예: Cloudflare 오렌지 구름)이 켜져 있으면 SSL 검증이 지연될 수 있으므로 초기 연결 시 주의

## 5. SSL 인증서 확인

Vercel은 도메인 연결이 정상적이면 SSL 인증서를 자동 발급합니다.

확인 순서:
1. Vercel `Domains` 화면에서 `Valid Configuration` 또는 유사한 정상 상태 확인
2. 브라우저에서 `https://대표도메인` 접속
3. 자물쇠 아이콘과 인증서 정상 여부 확인

## 6. 기본 도메인 리다이렉트

목표:
- `ey-vocal-homepage.vercel.app`
- 보조 도메인(`www` 또는 루트)

위 주소들이 모두 대표 도메인으로 정리되게 맞춥니다.

확인 항목:
- 기본 Vercel 주소 접속 시 대표 도메인으로 이동하는지
- `www`/루트 중 보조 주소 접속 시 대표 도메인으로 이동하는지

## 7. 연결 후 점검

- 데스크톱/모바일에서 대표 도메인 접속 확인
- 로고/이미지/폰트 깨짐 없음 확인
- 카카오톡 상담 버튼, 전화 링크, 외부 지도 링크 동작 확인
- `https` 강제 적용 여부 확인

## 8. 연결 후 후속 작업

도메인 연결이 끝나면 아래 항목을 이어서 진행합니다.

- `sitemap.xml`의 기본 URL을 커스텀 도메인으로 갱신
- `robots.txt`의 Sitemap 경로를 커스텀 도메인 기준으로 갱신
- Google Search Console 등록
- 네이버 서치어드바이저 등록

## 9. 성표님 체크리스트

- 대표 도메인 확정
- Vercel `Domains`에서 도메인 추가
- DNS 레코드 등록
- SSL 발급 확인
- 대표 도메인 리다이렉트 확인
- 도메인 연결 완료 후 봇 팀에 URL 전달
