# EY보컬스튜디오 콘텐츠 교체 가이드

대상 프로젝트: `ey-vocal-homepage`

본 가이드는 콘텐츠 placeholder(후기, 강사 텍스트, Hero 사진) 교체 범위만 다루며, 기능·자산 placeholder(지도, 예약, OG 이미지)는 범위 밖입니다.

현재 기준:
- 실제 사용 중인 사진 영역은 Hero 배경 1건입니다.
- 후기 4건은 모두 placeholder 상태입니다.
- 강사 사진과 Programs 사진은 현재 사이트에서 사용하지 않습니다.

## 1. 후기 교체 가이드

교체 위치:
- `#testimonials` 섹션 후기 4건
- 현재 각 후기 카드에는 `data-placeholder="true"`가 들어가 있습니다.

작성 형식:
- 이름 표기: `김○○ · 취미 수강 (3개월)`
- 본문 길이: 1~3줄
- 핵심 내용: 느낀 변화 1가지 + 수업 방식 체감 1가지

권장 템플릿:

```text
홍○○ · 취미 수강 (3개월)
높은 음에서 힘이 들어가던 습관이 줄었습니다.
레슨 뒤에 연습 포인트를 분명하게 정리해 주셔서 혼자 복습하기도 편했습니다.
```

작성 기준:
- 합쇼체를 유지합니다.
- 과장보다 실제 체감 중심으로 씁니다.
- 한 후기 안에 변화 포인트를 너무 많이 넣지 않습니다.
- 음정, 호흡, 발성, 연습 습관, 자신감 같은 구체 표현을 우선합니다.

금지어:
- `최고`
- `유일`
- `압도적`
- `무조건`
- `완벽`

성표님 수집 체크리스트:
- 이름 이니셜
- 수강 기간
- 레슨 전 고민 1개
- 달라진 점 1개
- 레슨 방식에서 좋았던 점 1개

## 2. 강사 정보 텍스트 교체

교체 위치:
- `#about` 섹션 내부 `about-editorial`
- `#instructor` 블록

현재 구조:
- 스튜디오 소개 문단
- 강사 소개 문단
- credential 리스트
- 추가 안내 펼침 영역

톤 가이드:
- 합쇼체 유지
- 과장 없이 단정하게 작성
- 경력 나열보다 수업 방식 설명을 우선
- 처음 보는 사람이 읽어도 바로 이해되는 문장 길이 유지

권장 문장 방향:
- 무엇을 가르치는지
- 어떤 기준으로 수업을 설계하는지
- 어떤 수강생에게 잘 맞는지

권장 예시:

```text
현재 목소리와 목표를 먼저 함께 정리하고, 그 기준으로 수업 흐름을 맞춥니다.
기본기와 곡 연습을 따로 떼지 않고, 필요한 순서대로 자연스럽게 연결합니다.
```

피해야 할 방향:
- 추상적인 수식어만 길게 반복하는 문장
- 검증하기 어려운 단정 표현
- 지나치게 광고성인 문장

## 3. Hero 배경 사진 교체

현재 사용 파일:
- `assets/images/studio-hero.webp`
- `assets/images/studio-hero.jpg`

권장 스펙:

| 항목 | 권장값 |
| --- | --- |
| 비율 | 2:3 세로형 |
| 기본 크기 | 1600×2400 |
| 포맷 | WebP + JPG 동시 준비 |
| 용량 | 각 파일 200KB 이하 권장 |
| 구도 | 인물 또는 공간 중심, 좌하단 텍스트가 겹쳐도 읽히는 여백 확보 |

교체 체크리스트:
- 너무 밝은 사진은 피합니다.
- 좌하단 텍스트가 묻히지 않도록 어두운 여백이 있는 사진을 우선합니다.
- 얼굴이나 장비가 화면 가장자리에 너무 타이트하게 붙지 않게 준비합니다.
- WebP와 JPG를 같은 구도로 맞춥니다.

## 4. 향후 추가용 사진 권장 스펙

주의:
- 아래 항목은 현재 사이트에서 사용하지 않습니다.
- 추후 강사 사진이나 Programs 비주얼을 다시 넣을 때만 참고합니다.

강사 프로필 사진:

| 항목 | 권장값 |
| --- | --- |
| 비율 | 2:3 세로형 |
| 권장 크기 | 800×1200 또는 1200×1800 |
| 포맷 | WebP + JPG |
| 용량 | 각 파일 200KB 이하 권장 |
| 구도 | 상반신 중심, 시선과 표정이 자연스러운 컷 |

Programs 비주얼:

| 항목 | 권장값 |
| --- | --- |
| 비율 | 3:2 가로형 |
| 권장 크기 | 1200×800 |
| 포맷 | WebP + JPG |
| 용량 | 각 파일 200KB 이하 권장 |
| 구도 | 악보, 마이크, 수업 장면 등 설명 없이도 레슨 맥락이 보이는 컷 |

## 5. 최종 확인 항목

- 후기 4건의 이름 형식이 통일됐는지 확인
- 후기 문장이 모두 합쇼체인지 확인
- 강사 소개 문구에 과장 표현이 없는지 확인
- Hero WebP/JPG 두 파일이 같은 장면인지 확인
- 새 사진 교체 후에도 텍스트 가독성이 유지되는지 확인
