## 호출 규칙
- IF: 상대 봇이나 사람을 부를 때 → THEN: 디스코드 @멘션만 사용한다.
- IF: 코덱봇이 디스코드 메시지를 받았고 자기 @멘션이 없다 → THEN: 응답하지 않는다.
- IF: 코덱봇이 클코봇 메시지를 받았다 → THEN: 독립 판단과 근거가 있을 때만 응답한다.
- IF: 사람 호출 사유가 실거래·자금·권한·API키·직접 확인 결과 공유가 아니다 → THEN: 봇끼리 해결한다.
- IF: 총괄봇에게 보고·승인·질문한다 → THEN: `<@1489117811652431882>` 멘션을 반드시 포함한다.
- IF: 받은 채널과 다른 채널에 답하려 한다 → THEN: 금지한다. 예외: 총괄 라우팅, 로그 기록(`CLAUDE.md` 기준), 크로스팀 `[결론]` 후 자기 작업방 `[크로스 결과]` 1회 공유.

## 안전장치
- IF: 현재 브랜치가 `codex/*`가 아니다 → THEN: 작업하지 않는다.
- IF: 구조·카피·디자인을 바꾼다 → THEN: `FINAL-DESIGN-DIRECTION.md`와 `FINAL-QA-GUARDRAILS.md` 기준을 먼저 확인한다.
- IF: HTML/CSS/JS를 바꿨다 → THEN: `python3 -m http.server 8000` 또는 `npx serve .`로 열고 데스크톱·모바일 레이아웃과 브라우저 콘솔 오류를 확인한다.
- IF: 외부 런타임 의존성이나 CDN 스크립트를 추가하려 한다 → THEN: 금지한다.
- IF: 아래 보호 파일 수정 요청을 받았다 → THEN: 수정하지 않는다.
